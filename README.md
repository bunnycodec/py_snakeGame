# py_snakeGame

## Purpose

Snake and Apple game made with python.

## Execution (Terminal)

```
python3 main.py
```

## Constraints

- Windows directory has .exe file which will run the game in Windows system
- for Linux / Mac `python3 main.py`
- use virtual env and install requirements.txt

### Python Dependencies

```
pip install pygame
```
